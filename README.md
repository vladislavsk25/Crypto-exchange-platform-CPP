# Crypto-exchange-platform-C-
This C++ implementation is a simulated cryptocurrency exchange platform that provides a realistic trading environment with order book management, wallet functionality, and market analysis tools.

#  Core Features<br/>


## Order Book System
**Two-way order matching:** Processes both bids and offers with appropriate price and time priority<br/>
**Real-time market data:** Monitors prices, volumes and market statistics<br/>
**CSV data integration:** Processes historical trading data from CSV files<br/>
**Product management:** Supports multiple cryptocurrency pairs (e.g. ETH/BTC)<br/>

## Trading Engine
**Order processing:** Inserting and managing buy/sell orders<br/>
**Automated matching:** Matches bid and ask prices using a price-time priority algorithm<br/>
**Trade execution:** Processes completed transactions and updates participants' wallets<br/>
**Sales tracking:** Records all concluded transactions with price and volume data<br/>

## Wallet Management
**Multi-currency support:** Tracks balances in different cryptocurrencies<br/>
**Automated settlement:** Processes trade settlements and updates balances<br/>
**Balance reporting:** Provides a complete overview of your wallet status<br/>

## User Interface
**Interactive menu system:** Console interface for trading operations<br/>
**Market analytics:** Displays key statistics (high/low prices, averages, volume)<br/>
**Portfolio management:** View wallet contents and trading results<br/>
**Time progression:** Simulates market movements on different timeframes<br/>
